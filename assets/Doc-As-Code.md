# Markdown Doc-as-Code

## Overview

This sample showcases a Doc-as-Code project created using Markdown and GitHub, featuring a Getting Started Guide and task-based documentation for user onboarding.

The documentation focuses on clear content structure, user-friendly instructions, and modern documentation workflows using version control and GitHub-based content management.

# Getting Started Guide


## Overview

This sample showcases a Getting Started Guide created for an Amazon AWS-based product to support new user onboarding and product setup.

The documentation focused on simplifying technical concepts, improving content structure, and creating a user-friendly onboarding experience. AI-assisted tools were used to understand user personas, organize the Table of Contents (TOC), and improve content flow.

- Instructional writing
- Documentation structuring
- AI-assisted content planning
